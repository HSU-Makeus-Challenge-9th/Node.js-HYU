## 🤙🏼 컨벤션
### 💡커밋 메시지
✅ **feat: Create Week{주차} Mission{번호}**

ex) 1주차 미션 1 완료한 경우
<br>
`feat: Create Week1 Mission1`

<br>

✅ **{커밋 태그}: {메시지}**

ex) `feat: 댓글 등록 기능 추가`


<br>

### 💫 commit 태그

| 태그 | 설명 |
| --- | --- |
| feat | 새로운 기능 추가 |
| fix | 버그 수정 |
| docs | 문서 수정 |
| style | 코드 스타일 수정 |
| refactor | 코드 리팩토링 |
| design | UI, CSS 수정 및 디자인 변경 |
| comment | 주석 추가(코드 변경 X) 혹은 오타 수정 |
| rename | 파일, 폴더명 수정 혹은 이동 |
| remove | 파일, 폴더 삭제 |
| chore | 초기 설정 및 수정 |
