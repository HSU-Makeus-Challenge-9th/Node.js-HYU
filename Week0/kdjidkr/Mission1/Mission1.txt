너디너리 홈페이지 접속하는 과정 적어보기 (소켓프로그래밍과 같은 개념 없이 TCP, IP, PORT 등의 개념 등 오늘 배운 내용으로 간단하게 적어주세요)
: 너디너리 홈페이지의 도메인은 다음과 같습니다. httpsdemoday.neordinary.co.kr 제가 이 url의 하이퍼링크를 누르거나 도메인창에 입력하고 접속하면, DNS 서버에 해당 도메인에 대응되는 IP주소를 전달 받을 수 있게됩니다. IP주소를 얻으면 제 브라우저는 TCP를 이용하여 Destination IP를 DNS에서 획득한 IP 주소로 TCP연결을 시도할 것입니다. 즉, 3 way handshake를 시도합니다. 이때, 너디너리홈페이지는 HTTPS를 사용하므로 PORT 번호는 443번입니다. 3way handshake 를 통해 클라이언트 - 서버 간 연결이 끝나면, 브라우저는 웹사이트를 보여주는 웹페이지를 요청하는 HTTP GET 메시지를 보낼 것이고 서버가 이에 응답하여 웹사이트 리소스를 전송할 것입니다.
